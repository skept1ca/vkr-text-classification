# Классификация текстовых документов на основе контекстуальных векторных представлений

**Выпускная квалификационная работа**

---

## Тема

Методы классификации текстовых документов на основе их контекстуальных векторных представлений в условиях ограниченных ресурсов

---

## Описание задачи

Многоклассовая классификация деловых документов (писем, служебных записок) по организационным подразделениям компании. Задача — автоматически определить получателя документа на основе его текстового содержания.

**Данные:** 1774 документа, 36 классов, тексты деперсонализированы (персональные данные заменены масками вида `[PERSON]`, `[ORGANIZATION]` и др.)

## Основные находки EDA

- Сильный дисбаланс классов: от 2 до 249 документов на класс
- 8 классов содержат менее 10 документов
- Медианная длина текста ~1278 символов (~250 слов)
- UMAP-визуализация показала семантическую близость классов — тексты разных классов не образуют чётких кластеров, что указывает на сложность задачи
